About Me

My name is Roman

I am a Full-Stack web developer from Ukraine. My interest in website design started back in the 10th grade of school, and since then, I have dedicated myself to web application development. I pursued my education in Computer Science and Technologies at Kharkiv National University of Radio Electronics (KhNURE). During my studies, I delved deeper into web development, learning new stages of development and technologies to build robust web applications. Consequently, all my projects undergo stages of business analysis, system architecture design, technical implementation, and web interface development. You can find examples of my work below. I am always on the lookout for exciting projects and challenges, so if you have an idea that you want to bring to life using web technologies, I will be delighted to help!

Front-End Development
I specialize in implementing web interfaces and related technologies such as HTML5 (hypertext markup language), CSS3 (cascading style sheets), JavaScript, and TypeScript. This front-end technology stack enables me to create flexible and straightforward solutions.
Back-End Development
For web applications' business logic, I use technologies like relational databases (MySQL & Postgres), the TypeScript programming language with Node.js runtime, and the NestJS framework. I also have experience in developing web applications using Java, utilizing the Spring framework, and can use the Express framework instead of NestJS. I do not work with non-relational databases.
Full-Stack Development
I am proficient in developing complete web applications, employing both back-end and front-end approaches and technologies. This allows me to save resources by handling both aspects of web development. My primary focus is on three-tier architecture (Database Server - Business Logic Server - Client), but I also have some experience in microservices architecture.

Web Applications

Fitness Club System

As a comprehensive full-stack endeavor, this project embodies an information management system tailored for fitness clubs. The core components of this system include a MySQL database with multiple tables and intricate relationships. The development process encompassed both the back-end and front-end aspects, employing cutting-edge technologies and frameworks to deliver a seamless and secure user experience. The back-end was meticulously crafted on the Node.js platform, utilizing the modern NestJS framework. This framework's modular architecture and object-oriented programming (OOP) patterns were leveraged to create a scalable and maintainable system. Security and privacy were paramount considerations, leading to the incorporation of JWT (JSON Web Token) technology and the bcrypt library, safeguarding sensitive user data. Angular emerged as the preferred front-end platform, enabling the creation of dynamic web pages in the form of reusable components. These components synergistically interacted to deliver a user-friendly and visually engaging interface. The interface design catered to the needs of fitness club members, streamlining their interaction with the system. A standout feature of this application was the cleverly implemented recommendation questionnaire, designed to guide users towards selecting the most suitable membership plan. Functioning as a production model of knowledge, this feature employed SQL queries to dynamically adjust the recommended membership plans based on user responses. As users progressed through the questionnaire, the system presented progressively tailored information derived from these queries, ensuring an informed and confident decision-making process.
